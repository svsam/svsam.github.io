## Website applications

To refresh the generator, build it in the
[`ASCIIArtGenerator`](https://github.com/svsam/ASCIIArtGenerator) source
repository with `npm run build:web`, then replace the deployable files in
`web/` with the contents of that project's `dist/web/`. Keep the generated
`web/pkg/` files versioned; the browser worker needs them at runtime.
